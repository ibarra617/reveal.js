What is GDevelop?

  • Open-source, cross-platform game engine
  • No programming skills required
  • Uses visual scripting and event-based system
  • Supports 2D game development

Non-Code Contribution Opportunities

  • Areas for Non-Code Contributions:
    • Documentation
    • Community Support
    • Asset Creation
    • Testing features
    • Marketing and Visual Aspects
  • Links to Open Issue(s) or Examples:
    • Typo and Wording Fixes Issue on GDevelop
  • Why Are These Contributions Valuable?
    • Improve user experience
    • Expand the user base
    • Enhance software quality

Contribution Guidelines

  • The guidelines can be found in the CONTRIBUTING.md file in the GDevelop GitHub repository.
  • The onboarding process is extremely beginner friendly and encourages new contributors to get involved with easy simple contributions.
  • The only thing that surprised me was how beginner friendly and encouraging this repository is for new contributors.It creates a welcoming and easy to get started environment.
